    Instal·lació i habilitació del SNMP a Ubuntu i Windows

    1. Instal·lació i configuració del SNMP en Ubuntu:

    SNMP (Simple Network Management Protocol) és un protocol utilitzat per monitoritzar dispositius de xarxa com routers, switches i servidors. En aquest cas, explicarem com instal·lar i habilitar SNMP en un sistema Ubuntu per poder obtenir dades de monitorització.

    Passos per a Ubuntu:

    1. Instal·la els paquets SNMP:
    2. En primer lloc, actualitza el sistema i instal·la els paquets necessaris per utilitzar SNMP:
    Bash
    1
2
    sudo apt update
sudo apt install snmp snmpd
    1. Configura el servei SNMP:
    2. El servei SNMP es configura mitjançant el fitxer snmpd.conf, que es troba a la ruta /etc/snmp/snmpd.conf. Abans de modificar-lo, és aconsellable fer una còpia de seguretat del fitxer:
    Bash
    1
    sudo cp /etc/snmp/snmpd.conf /etc/snmp/snmpd.conf.bak
    1. Modifica la configuració de snmpd.conf:
    2. Obre el fitxer de configuració per editar-lo:
    Bash
    1
    sudo nano /etc/snmp/snmpd.conf
    • Modifica o afegeix les línies següents per establir el nom de comunitat (usarem "public" com a exemple):
    Bash
    1
2
3
4
    # Definir la comunitat SNMP
rocommunity public
# Definir l'agent SNMP per escoltar a totes les interfícies de xarxa
agentAddress udp:161,udp6:[::1]:161

    • Si vols afegir altres configuracions personalitzades, pots fer-ho, però aquests són els valors mínims per a la configuració bàsica.

    • Reinicia el servei SNMP:

    • Un cop fetes les modificacions, reinicia el servei SNMP per aplicar els canvis:
    Bash
    1
    sudo systemctl restart snmpd
    1. Habilita el servei SNMP per començar a arrencar automàticament:
    2. Si vols que el servei SNMP s’iniciï automàticament quan el sistema arrenca, utilitza:
    Bash
    1
    sudo systemctl enable snmpd
    1. Comprova que el servei SNMP estigui funcionant:
    2. Per verificar que el servei SNMP està en funcionament, pots executar:
    Bash
    1
    snmpwalk -v 2c -c public localhost

    Aquesta comanda enviarà una sol·licitud SNMP a la màquina local i et retornarà informació sobre els dispositius SNMP de la màquina.

    2. Instal·lació i configuració del SNMP en Windows:

    A diferència d'Ubuntu, el sistema operatiu Windows no té SNMP instal·lat per defecte, així que cal afegir el servei de SNMP manualment.

    Passos per a Windows:

    1. Instal·la el servei SNMP:

    2. A Windows 10 o Windows Server 2016 i posteriors, el servei SNMP es troba com a característiques opcionals que cal instal·lar.

    3. A continuació, anem a "Panell de control" > "Programari i característiques" > "Activar o desactivar les característiques de Windows".
    4. Busca i marca l’opció "Servidor SNMP" i les seves subopcions "Comunitat SNMP" i "Agent SNMP". Fes clic a "D'acord".

    Un altre mètode seria instal·lar les característiques des de PowerShell:

    • Obrir PowerShell com a administrador i executar:
    PowerShell
    1
    Enable-WindowsOptionalFeature -Online -FeatureName SNMP

    1. Configura SNMP:

    2. Un cop instal·lat el servei, anem a "Panell de control" > "Eines administratives" > "Serveis".

    3. Busca el servei "SNMP Service" a la llista, i fes clic amb el botó dret sobre ell.

    4. Fes clic a "Propietats" i selecciona "Iniciat" a l’opció Tipus d’arrencada. Configura també el servei per a que s'iniciï automàticament.

    5. Configura les comunitats SNMP:

    6. Fes clic a la pestanya "Seguretat" i defineix la comunitat SNMP.

    7. A la secció "Comunitats", fes clic a "Afegir" per definir una nova comunitat SNMP.

    8. Per exemple, afegeix la comunitat "public" per la lectura de dades.

    9. Configura les adreces IP de la màquina que poden consultar SNMP:

    10. En el mateix menú de "Seguretat", a la secció "Permetre sol·licituds SNMP només des de les adreces IP seleccionades", pots especificar les IPs de les màquines que podran consultar SNMP (per exemple, només permetre consultes des de la teva xarxa local).

    11. Reinicia el servei SNMP:

    12. Després d'haver configurat les opcions, reinicia el servei SNMP per aplicar els canvis. Pots fer-ho des del mateix menú de "Serveis" fent clic a "Reiniciar" o amb PowerShell:

    PowerShell
    1
    Restart-Service SNMP

    1. Comprova que el servei SNMP estigui funcionant:

    2. Un cop configurat, pots comprovar el funcionament de SNMP mitjançant eines com snmpwalk o un client SNMP, per exemple, des d’un altre dispositiu (Linux, Windows o una eina de monitoratge).

    Bash
    1
    snmpwalk -v 2c -c public <IP_DEL_COMPUTADOR_WINDOWS>

    Comprovació

    • Per a Ubuntu: Si el servei està correcte, pots fer un snmpwalk a la mateixa màquina o des d'una altra màquina per verificar que els dispositius responen correctament.

    • Per a Windows: Després de configurar SNMP, pots provar amb eines com snmpwalk o utilitzar eines com The Dude o PRTG Network Monitor per verificar que SNMP està responent correctament.
