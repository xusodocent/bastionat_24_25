    Crear appliances

    Mikrotik

    1. Appliance

    Baixa alguna versió estable de la web de Mikrotik en la secció Cloud hosted routers i tipus Raw disk image.

    Fem una màquina Qemu nova.

    Selecciona la imatge baixada.

    Canvía el símbol.

    2. Configuració

    Podem configurar Mikrotik de tres formes diferents.

    • Amb Winbox
    • Amb un appliance de GNS3
    • Navegador web

    Warning

    L'aplicació de Winbox està disponible per a Windows, si vols utilitzar-la en Linux has de fer ús de Wine.

    Warning

    Per a poder utilitzar el navegador cal que router tinga alguna adreça IP i que hi haja enrutament fins la màquina on utilitzes el navegador.

    3. Xarxa bàsica

    El següent esquema correspon a una xarxa bàsica en la que els ordinadors poden navegar.

    Cal configurar el Mikrotik per a poder navegar, per tant haurem de configurar les adreces de la part LAN, activar un servidor dhcp i fer una regla en el firewall de tipus masquerade per a que el router puga fer NAT.

    Fem la configuració amb Winbox que després explicarem com funciona.

    Canvia el nom de les interfícies del router a WAN i LAN.

    Observa com en la interfície WAN s'ha creat automàticament un dhcpclient i obté l'adreça per dhcp. Configurem la LAN amb 10.0.0.1/24

    Fem el server dhcp directament des del setup

    Indiquem amb una regla de firewall de tipus NAT que a tot tràfic que vaja a eixir per la WAN se li aplique un masquerade.

    Comprovem que assigne les adreces correctes...

    Bash
    1
2
    PC1> ip dhcp
DORA IP 10.0.0.253/24 GW 10.0.0.1

    Des de l'appliance de Firefox es veu com es pot navegar.

    Appliance Ubuntu Server

    Per al punt 1 haurem de crear una màquina virtual en Virtual Box, o almenys hem de tenir un disc vdi o mdk

    1. Crear appliance des d'un disc dur de VirtualBox

    Primer -> Tindre un disc creat en Virtualbox o Vmware

    Segon -> Seleccionar New Template. Crear de forma manual

    Tercer -> Ens obrirà el menú de preferències i triarem l'apartat QEMU

    Quart -> Nom de la nova plantilla

    Cinqué -> Triar el virtualitzador, en aquesta cas qemu. Ho deixarem per defecte. I triar la quantitat de memòria RAM per a la màquina.

    Sisé -> Ara triem el tipus de consola. En el nostre cas vnc.

    Seté -> En aquest apartat triem New imatge i seleccionem el disc vdi o mdk

    Warning

    IMPORTANT -- Sembla que es quede penjat però està important la màquina.

    Warning

    PACIÈNCIA

    Huité -> Seguidament ens pregunta si volem copiar-la al directori d'imatges. Li diguem que Si.

    Nové -> Apliquem els canvis i ja tenim la plantilla disponible per a utilitzar.

    2. Crear des del GNS3 Marketplace

    El procediment és el mateix que per al firefox, la única diferència és que en el Guest triarem un Ubuntu Server. Ens rediriguirà a la pàgina d'osboxes. Un lloc on podem triar les màquinas ja instal·lades per a descarregar. La diferència és que les versions no coincidixen i haurem de crear una versió nova i triar la imatge que hem descarregat.

    Warning

    Es descarrega un 7zip que hem de descomprimir i ja trindrem la imatge del disc per a importar.
