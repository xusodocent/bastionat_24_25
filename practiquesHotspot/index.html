    Hotspot amb Mikrotik

    Ací tens la guia per configurar un Hotspot a la interfície ether3 (anomenada HOTSPOT) amb IP 10.0.1.1/24, usuaris locals, i amb certificat creat prèviament (imprescindible per al funcionament complet).

    ✅ 1. Canviar el nom de la interfície a HOTSPOT

    📍 Interfaces > selecciona ether3 > canvia el nom a HOTSPOT

    O via terminal:

    Bash
    1
    /interface set [find default-name=ether3] name=HOTSPOT

    ✅ 2. Assignar IP a la interfície

    📍 IP > Addresses > +

    • Address: 10.0.1.1/24
    • Interface: HOTSPOT

    O via terminal:

    Bash
    1
    /ip address add address=10.0.1.1/24 interface=HOTSPOT

    ✅ 3. Crear un certificat SSL

    📍 System > Certificates > +

    1. Crea el certificat:
    Bash
    1
    /certificate add name=hotspot-cert common-name=hotspot.mikrotik.local key-size=2048 days-valid=3650
    1. Signa’l:
    Bash
    1
    /certificate sign hotspot-cert
    1. Activa’l:
    Bash
    1
    /certificate set hotspot-cert trusted=yes

    💡 Espera uns segons que el certificat estigui en estat KT (Key+Trusted)

    ✅ 4. Crear el servidor DHCP per la interfície HOTSPOT

    📍 IP > DHCP Server > DHCP Setup

    • Interface: HOTSPOT
    • IP Range: 10.0.1.10 - 10.0.1.100
    • Gateway: 10.0.1.1
    • DNS: 1.1.1.1 (o el que vulguis)

    ✅ 5. Crear el Hotspot

    📍 IP > Hotspot > Hotspot Setup

    • Interface: HOTSPOT
    • IP: 10.0.1.1/24
    • Address Pool: 10.0.1.10-10.0.1.100
    • Certificate: selecciona hotspot-cert
    • SMTP Server: pots deixar-ho en blanc
    • DNS Name: hotspot.mikrotik.local
    • User: pots ometre’l si crearàs usuaris personalitzats

    ✅ 6. Crear usuaris locals

    📍 IP > Hotspot > Users > +

    Exemple:

    Name Password Profile
    usuari_ciber1 abc123 default
    usuari_ciber2 xyz456 default

    Via terminal:

    Bash
    1
2
    /ip hotspot user add name=usuari_ciber1 password=abc123 profile=default
/ip hotspot user add name=usuari_ciber2 password=xyz456 profile=default

    ✅ 7. Comprovar funcionament

    1. Connecta un client a la interfície HOTSPOT
    2. El dispositiu ha de rebre IP del rang 10.0.1.x
    3. En accedir a qualsevol web, es redirigirà a https://hotspot.mikrotik.local/login
    4. Introduir usuari i contrasenya → accés a Internet

    📥 Com importar-lo

    1. Obre Winbox o WebFig
    2. Ves a Files
    3. Crea un fitxer nou anomenat: fitxes_hotspot.rsc i enganxa aquest contingut
    4. O bé, crea el fitxer .rsc amb un editor de text i arrossega'l dins Winbox
    5. Ves a New Terminal i executa:
    Bash
    1
    /import file-name=fitxes_hotspot.rsc

    Comprovacions

    🔁 Recorda: assegura’t que el perfil 2h-limit existeix a IP > Hotspot > User Profiles, amb el session-timeout=2h i opcionalment rate-limit.
