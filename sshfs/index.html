    SSHFS

    SSHFS

    SSHFS (Secure SHell FileSystem) és un sistema de fitxers basat en FUSE (Filesystem in Userspace) que permet muntar un directori remot a través de SSH i treballar amb ell com si fos part del sistema de fitxers local. Amb SSHFS, pots accedir, modificar i gestionar fitxers d'una màquina remota sense necessitat de copiar-los al sistema local, utilitzant el protocol de Secure Shell (SSH) per garantir la seguretat.

    Pràctica: Muntar un SSH Filesystem remot i accedir des de màquines clients

    Objectius

    • Configurar un servidor SSH per permetre l'accés a fitxers de manera remota.
    • Muntar el sistema de fitxers remot en una màquina client utilitzant SSHFS.
    • Treballar amb fitxers remots com si estigueren en un directori local.

    Requeriments previs

    Servidor remot:

    • Accés a un servidor amb SSH configurat.
    • Instal·lat el paquet necessari per al servei SSH (habitualment openssh-server).

    Client:

    • Sistema operatiu Linux/Unix.
    • Paquet sshfs instal·lat.

    Procediment

    Configurar el servidor remot

    • Assegurar-se que el servei SSH està actiu:

    Instal·lar el servidor SSH si no està disponible:

    Bash
    1
2
    sudo apt update
sudo apt install openssh-server

    Activar i assegurar-se que el servei està funcionant:

    Bash
    1
2
3
    sudo systemctl enable ssh
sudo systemctl start ssh
sudo systemctl status ssh
    • Crear un usuari per a accés remot:
    Bash
    1
    sudo adduser usuari_remot
    • Configurar permisos i seguretat (opcional):

    Editar el fitxer /etc/ssh/sshd_config per assegurar-se que permet connexions:

    Text Only
    1
2
    PermitRootLogin no
PasswordAuthentication yes

    Reiniciar el servei SSH per aplicar els canvis:

    Bash
    1
    sudo systemctl restart ssh

    Instal·lar i configurar SSHFS al client

    • Instal·lar SSHFS:
    Bash
    1
2
    sudo apt update
sudo apt install sshfs

    • Crear un punt de muntatge:

      • Triar un directori local on es muntarà el sistema de fitxers remot:

        Bash
        1
        mkdir -p ~/ssh_remot

    • Muntar el sistema de fitxers remot:

      • Utilitzar SSHFS per muntar el directori remot al punt local:

        Bash
        1
        sshfs usuari_remot@IP_SERVIDOR:/directori/remot ~/ssh_remot

      • Exemple:

        Bash
        1
        sshfs usuari_remot@192.168.1.100:/home/usuari_remot ~/ssh_remot

    Validar el muntatge

    Comprovar que el directori està muntat:

    Bash
    1
    df -h
    Hauria d'aparèixer una línia que indique el muntatge mitjançant SSHFS.

    Navegar pel sistema de fitxers remot com si fos local:

    Bash
    1
2
    cd ~/ssh_remot
ls

    Crear, editar i eliminar fitxers per comprovar l'accés:

    Bash
    1
2
3
    touch prova.txt
echo "Hola món remot!" > prova.txt
cat prova.txt

    ** Desmuntar el sistema de fitxers**

    • Una vegada finalitzada la sessió, desmuntar el directori:
    Bash
    1
    fusermount -u ~/ssh_remot

    Tasques complementàries

    Configurar claus SSH per evitar utilitzar contrasenyes repetides:

    Generar una clau SSH al client:

    Bash
    1
    ssh-keygen -t rsa -b 4096

    Copiar la clau pública al servidor:

    Bash
    1
    ssh-copy-id usuari_remot@IP_SERVIDOR
    • Fer el muntatge automàtic afegint una entrada a /etc/fstab (opcional i avançat).

    Muntatge de forma permanent

    FSTAB

    Utilitzar fstab permet muntar el sistema de fitxers SSHFS automàticament cada vegada que l'equip client inicia.

    Procediment

    Verificar que SSHFS funciona manualment

    Abans d'afegir una entrada a fstab, assegura't que pots muntar el sistema de fitxers remot manualment:

    Bash
    1
    sshfs usuari_remot@IP_SERVIDOR:/directori/remot ~/ssh_remot

    Si això funciona correctament, pots continuar amb la configuració a fstab.

    Crear un punt de muntatge permanent

    Defineix un directori en el sistema local que sempre serà el punt de muntatge:

    Bash
    1
    mkdir -p ~/ssh_remot

    Configurar les credencials (opcional, però recomanat)

    Hauràs realitzat abans aquest pas, però el posem ací també a mode de tutorial. Ens evitem haver d'introduir la contrasenya.

    • Genera una clau SSH al client:
    Bash
    1
    ssh-keygen -t rsa -b 4096
    • Copia la clau pública al servidor:
    Bash
    1
    ssh-copy-id usuari_remot@IP_SERVIDOR
    • Prova l'accés sense contrasenya:
    Bash
    1
    ssh usuari_remot@IP_SERVIDOR

    Modificar el fitxer /etc/fstab

    • Obre el fitxer /etc/fstab amb permisos d'administrador:
    Bash
    1
    sudo nano /etc/fstab
    • Afegeix una línia al fitxer per configurar el muntatge automàtic amb SSHFS:
    Text Only
    1
    usuari_remot@IP_SERVIDOR:/directori/remot  /home/usuari/ssh_remot  fuse.sshfs  defaults,_netdev,user,idmap=user,IdentityFile=/home/usuari/.ssh/id_rsa,allow_other  0  0

    usuari_remot: el nom de l'usuari al servidor remot.

    IP_SERVIDOR: l'adreça IP del servidor remot.

    /directori/remot: el directori al servidor remot que vols muntar.

    /home/usuari/ssh_remot: el punt de muntatge local.

    idmap=user: manté els permisos d'usuari.

    IdentityFile: especifica la ruta de la clau SSH (si l'utilitzes).

    allow_other: permet que altres usuaris al sistema accedeixin al muntatge (opcional).

    Provar el muntatge

    Força el muntatge de l'entrada a fstab sense reiniciar:

    Bash
    1
    sudo mount -a

    Comprova si el sistema de fitxers està muntat:

    Bash
    1
    df -h

    Resoldre problemes comuns

    • Si tens problemes perquè el muntatge depèn de la connexió de xarxa, afegeix l'opció _netdev a l'entrada de fstab. Aquesta opció assegura que el muntatge només es fa quan la xarxa està activa:
    Text Only
    1
    defaults,_netdev
    • Si l'usuari no té permisos per muntar el sistema de fitxers, afegeix l'opció user per permetre als usuaris no root muntar-lo:
    Text Only
    1
    defaults,_netdev,user

    allow-other

    • Recorda que allow_other només funcionarà si tens habilitat aquesta opció a FUSE.

    Edita /etc/fuse.conf i descomenta:

    Text Only
    1
    user_allow_other

    Resultat

    Després de configurar fstab, el sistema muntarà automàticament el directori remot mitjançant SSHFS en cada arrencada, i el punt de muntatge estarà disponible al directori definit. Això és útil per garantir un accés remot consistent i automatitzat als recursos del servidor.
